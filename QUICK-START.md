## 🚀 Three Ways to Run

### 1. **Direct File Open (Easiest)**
1. Find `index.html` in the project folder
2. **Double-click it** to open in your default browser
3. 🎉 **Done!** App is running

✅ **Works offline** | ✅ **No installation needed** | ✅ **All data stays local**

---

### 2. **Python HTTP Server (Recommended)**
```bash
# Navigate to project folder
cd path/to/studyhub

# Start server
python -m http.server 8000

# Open browser to http://localhost:8000
```

---

### 3. **Node.js HTTP Server**
```bash
# Install if you don't have it
npm install -g http-server

# Run in project folder
http-server

# Visit the displayed URL (usually http://localhost:8080)
```

## 📚 First Steps

1. **Select a Subject** from the dropdown
   - Calculus I, II, III
   - Differential Equations
   - Measure Theory
   - Chemistry

2. **Click "Generate Problem"**
   - Random problem appears with 4 options
   - 1 correct answer, 3 distractors

3. **Pick Your Answer**
   - Select one of the 4 options
   - Get instant feedback
   - See which answer is correct

4. **View Solution**
   - Click "Reveal Step-by-Step Solution"
   - Read detailed explanation with math
   - Learn how to solve it

5. **Use the Sketchbook**
   - Draw diagrams alongside problems
   - Add text notes
   - Download sketches as PNG files

## 🆘 Having Issues?

### Canvas Drawing Not Working?
- Make sure you've selected a tool (Pen or Eraser)
- Try in a different browser

### Math Not Displaying?
- Wait a moment - MathJax is rendering
- Try refreshing the page
- Check browser console for errors

### Sketches Not Saving?
- Check browser's localStorage is enabled
- Try clearing browser cache

### Slow Performance?
- Close other tabs
- Use a smaller brush size
- Try a more recent browser
